# COLOR_VALUE

A
[COLOR_VALUE](https://opensource.hcltechsw.com/domino-c-api-docs/reference/Data/COLOR_VALUE/).

## Examples
```yaml
Flags: 0x0001
rgb: 0xffff00
```

## Required properties

### `Flags`
* type: number
* see:
[COLOR_VALUE](https://opensource.hcltechsw.com/domino-c-api-docs/reference/Data/COLOR_VALUE/)

### `rgb`
32-bit rgb value (0xRRGGBB)
* type: number
* see:
[COLOR_VALUE](https://opensource.hcltechsw.com/domino-c-api-docs/reference/Data/COLOR_VALUE/)
